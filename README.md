# Boilerworks FastAPI Micro

> Lightweight async Python API service with FastAPI. No frontend.

**Status:** Planned

FastAPI API-only microservice for teams that need a fast, async Python backend without frontend overhead. Ideal for internal APIs, webhook handlers, data pipelines, ML model serving, and backend-for-frontend services. Minimal dependencies, automatic OpenAPI docs, and async-first by default.
